# Pengaruh Usia & Tekanan Darah Diastolik terhadap Penyakit Jantung Koroner (Regresi Logistik)

## Deskripsi
Penelitian ini menganalisis pengaruh usia dan tekanan darah diastolik terhadap penyakit jantung koroner menggunakan regresi logistik.

## Tools
- R
- Regresi Logistik

## Hasil Analisis
Lihat hasil lengkap di RPubs:
https://rpubs.com/reyginaww/1192206

## Kesimpulan
Hasil analisis menunjukkan bahwa usia dan tekanan darah diastolik berpengaruh signifikan terhadap kemungkinan terjadinya penyakit jantung koroner berdasarkan model regresi logistik.
